​Amazon Homepage Clone 🚀

​Hi everyone! I’m Rahul, and I am currently a student on an exciting journey into web development. Since completing my 12th-grade examinations, I have been dedicating my time to learning the fundamentals of coding.
​This project is my first attempt at replicating a professional-grade website. I chose to clone the Amazon homepage to practice my front-end skills and understand how complex layouts are built from scratch.

​🛠 What I learned
​This project helped me turn theory into practice:

​Structuring with HTML: I learned how to properly structure a webpage and organize content.

​Styling with CSS: 
I gained hands-on experience with CSS layout techniques, including the use of Flexbox to keep elements aligned and responsive.
​Web Development Workflow: I learned the end-to-end process of building a project, from analyzing the original website design to implementing features like navigation bars, search inputs, and image-based grids.
​Problem Solving: I faced challenges with spacing and layout alignment, which taught me how to analyze elements and debug styles in real-time.

​💡 Why I built this?
​As a beginner, I wanted to move beyond just reading tutorials and actually "get my hands dirty" by building something I use every day. This project is a milestone in my learning path, and I look forward to adding more functionality—like JavaScript—as I continue my studies
